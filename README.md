# Credit Card Validator

A C program that validates credit card numbers using **Luhn's Algorithm** and identifies the card type.

## Features

- Validates card numbers using Luhn's Algorithm
- Detects:
  - American Express (AMEX)
  - MasterCard
  - Visa
- Input validation

## Technologies

- C
- Standard Library (`stdio.h`, `stdbool.h`)

## How to Compile

```bash
gcc credit.c -o credit
```

This program is a refinement of "credit" problem in pset 1 of CS50's course, week 1.
